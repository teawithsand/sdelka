# sdelka

Sdelka is supposed to be anki-like SRS app.

Under the hood it uses [sdelka-engine](https://github.com/teawithsand/sdelka-engine).

# Building/Running
In order to build app:
1. Clone repo
1. Run `yarn`
1. Run `yarn develop`
1. It should run

## Testing
In order to run tests use `yarn test`. Not much magic here
